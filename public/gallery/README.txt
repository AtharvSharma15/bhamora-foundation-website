=====================================
     HOW TO ADD YOUR PHOTOS
=====================================

1. COPY your photos into this folder:
   public/gallery/

   Supported formats: .jpg, .jpeg, .png, .webp, .avif

2. OPEN the file below and follow the instructions inside it:
   src/content/galleryData.ts

   You need to add an entry for each photo with the filename
   you copied into this folder.

3. SAVE the file. The gallery will update automatically.


-------------------------------------
 OPTIONAL: Thumbnails (for speed)
-------------------------------------
- Create smaller versions of your images
- Place them in: public/gallery/thumbnails/
- Reference them as: /gallery/thumbnails/your-photo.jpg
